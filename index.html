WikiWalk

It's a well-known fact* that if you start at a random wikipedia article and continuously hit the first non-italicized, non-parenthesized link, you'll get to the page for Philosophy. This project lets you prove it about 97 percent of the time. Built using D3, one of JavaScript's premier data visualization libraries, WikiWalk is a visual stroll through Wikipedia.
